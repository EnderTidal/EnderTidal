## Frameworks

**Z-Based Design** — Start with the most powerful AI model available. Progressively shift to cheaper and more deterministic solutions as the problem stabilizes. Let intelligence find the shape, then harden it.

**Steersman Build Method** — Define. Delete. Ship. Measure. Trust + Release. Five steps. No sixth.

**Human Context Protocol** — Structured context delivery for AI systems. The difference between an AI that knows what to do and one that asks.
